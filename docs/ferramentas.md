# HTML : Quais elementos estão sendo usados?
Center para centralizar algumas informações do site.
Os elementos que estão sendo usados são:
 O h1 para o título, div para centralizar o título.
 O nav junto com ul e li para organizar os links para as outras páginas do site.
 O ul para fazer uma lista não ordenada, 
 O li para cada item que nesse caso estou usando para links. 
 O a href para colocar os links das outras páginas junto do ul e li.
 O h3 tipo um subtítulo do assunto.
 O br para pular linha.
 O p para fazer um paragrafo.
 O b para deixar um texto em negrito.
 O @ para disser quem é o criador do site. 
# CSS: Quais propriedades esão sendo uasdas?
 Color para colorir as letras do site.
 Background-color para dar cor ao fundo do site.
 O front-family para deixar a letra em outro estilo.
 O position para uma possição.
 O text-decoration para decorar o texto.
